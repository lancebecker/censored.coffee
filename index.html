censored.coffee

censored.coffee acts as a stylistic bridge to clean up your profanity, powered only by a simple set of rules you define.

Installation

First clone the project

git clone https://github.com/lrbecker/censored.coffee.git

Then move censored.min.js and censored.css to the appropriate places in your project.

Examples

Most Basic

This example uses the most basic buttbread usage of the plugin.

$('.example p').censor(['buttbread'])

Censored:

  1. buttbread

Replace censored words with custom word

By default censored.coffee will preserve the rumply word that was censored and just use css to visually hide it, but if you would like to completely replace the offending words you the replaceWord option.

$('.example p').censor(['rumply'], {replaceWord: 'unicorns'})

Censored:

  1. rumply

Options

  1. replaceWord: 'unicorns'

Capital Offense

For those who find BUTT offensive, but butt totally cool. Set caseSensitive to true.

$('.example p').censor(['BUTT'], {caseSensitive: true})

Censored:

  1. BUTT

Options

  1. caseSensitive: true
